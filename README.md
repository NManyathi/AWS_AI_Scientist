## 2️⃣ Dog Breed Classifier

### 📌 Overview

A computer vision project using pretrained Convolutional Neural Networks (CNNs) to classify:

* Dogs vs. not-dogs
* Specific dog breeds

The project compares the performance of:

* VGG
* ResNet
* AlexNet

### 📚 Key Concepts

* Transfer learning
* Image preprocessing
* CNN architectures
* Model evaluation
* Classification accuracy analysis

### 📊 Key Findings

* VGG achieved the strongest breed classification accuracy
* ResNet delivered strong overall performance
* AlexNet showed lower accuracy due to older architecture design

### 🛠️ Technologies Used

* Python
* PyTorch
* PIL
* NumPy
* Shell scripting

---

## 3️⃣ SentimentScope — IMDB Sentiment Analysis using Transformers

### 📌 Overview

A transformer-based Natural Language Processing (NLP) project that classifies IMDB movie reviews as:

* Positive
* Negative

The project implements a custom transformer architecture using PyTorch and Hugging Face tokenization workflows.

### 📚 Key Concepts

* NLP preprocessing
* Tokenization
* Transformer attention mechanisms
* Positional embeddings
* Binary text classification
* Deep learning model evaluation

### 📊 Final Results

| Metric              | Result |
| ------------------- | ------ |
| Validation Accuracy | 78.56% |
| Test Accuracy       | 76.40% |

### 🛠️ Technologies Used

* Python
* PyTorch
* Hugging Face Transformers
* Pandas
* Matplotlib
* CUDA GPU Training

# 🚀 How to Explore This Repository

## 1. Clone the Repository

```bash
git clone https://github.com/NManyathi/<repository-name>.git
cd AWS_AI_Scientist
```

---

## 2. Create a Virtual Environment (Recommended)

```bash
python -m venv aws_ai_env
```

Activate environment:

### Windows

```bash
aws_ai_env\Scripts\activate
```

### macOS/Linux

```bash
source aws_ai_env/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook or project folder to explore exercises, experiments, and mini-projects.
